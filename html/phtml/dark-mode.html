Add Dark Mode to Your Website

Dark mode improves user experience by reducing eye strain and offering a modern look. Implementing it with JavaScript and CSS is easy!

1. Add CSS Variables


:root {
  --bg: #ffffff;
  --text: #33415C;
}

.dark-mode {
  --bg: #121212;
  --text: #f5f5f5;
}

body {
  background-color: var(--bg);
  color: var(--text);
}

2. Toggle Button with JavaScript


const toggle = document.getElementById("darkToggle");
toggle.onclick = () => {
  document.body.classList.toggle("dark-mode");
};

3. Optional: Save Preference in LocalStorage


localStorage.setItem("theme", "dark"); // or "light"
