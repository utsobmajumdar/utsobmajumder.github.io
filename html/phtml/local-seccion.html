LocalStorage vs SessionStorage

Both LocalStorage and SessionStorage allow storing data in the browser, but they behave differently.

1. LocalStorage

Data persists even after the tab or browser is closed. Example:


localStorage.setItem("username", "Utsob");
const user = localStorage.getItem("username");

2. SessionStorage

Data is lost when the tab is closed:


sessionStorage.setItem("token", "abc123");

3. When to Use

  • LocalStorage: User preferences, themes, cart info.
  • SessionStorage: One-time logins, form state.
