How HTML Works

HTML (HyperText Markup Language) is the foundation of every website on the internet. It tells the browser what to display and how to structure content.

1. The Structure of HTML

HTML uses tags like <h1>, <p>, <div> to organize content into headings, paragraphs, sections, etc. These tags form a tree-like structure called the DOM (Document Object Model).

2. The Browser's Job

When you open a website, the browser receives HTML files from the server. It reads the tags line-by-line and renders the structure visually on your screen. It also downloads and applies CSS and JavaScript if available.

3. Tags & Elements

Each HTML element has an opening and closing tag. For example:

<p>This is a paragraph</p>

Some elements are self-closing, like <img /> or <br />.

4. Common Tags

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <a>: Links
  • <img>: Images
  • <div> / <section>: Containers
