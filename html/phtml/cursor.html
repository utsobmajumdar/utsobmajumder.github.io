Create a Custom Cursor with JavaScript

Want to make your website feel more interactive? Replace the boring default cursor with a custom one using HTML, CSS, and JS.

1. Create the Cursor Element


<div class="cursor"></div>

2. Style It


.cursor {
  width: 20px;
  height: 20px;
  border: 2px solid #FB8500;
  border-radius: 50%;
  position: fixed;
  pointer-events: none;
  transition: transform 0.1s ease;
}

3. Move It with JS


document.addEventListener("mousemove", (e) => {
  const cursor = document.querySelector(".cursor");
  cursor.style.left = e.clientX + "px";
  cursor.style.top = e.clientY + "px";
});
