Async & Await in JavaScript

Writing asynchronous code can get messy with nested callbacks. Thankfully, async/await makes it much cleaner.

1. Understanding Promises

A promise is an object that may return a value now or later. Example:


const fetchData = () => {
  return new Promise((resolve) => {
    setTimeout(() => resolve("Done!"), 1000);
  });
};

2. Using Async/Await

Instead of chaining .then(), you can write:


async function getData() {
  const result = await fetchData();
  console.log(result);
}

3. Error Handling

Use try...catch to handle errors gracefully:


try {
  await someAsyncCall();
} catch (error) {
  console.error(error);
}

⚙️ Cleaner async code = better developer experience!
